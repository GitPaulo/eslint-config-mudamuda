# Installation & Usage
`npm install --dev eslint-config-mudamuda`

Afterwards, add the config to `eslintrc.js` or `eslintrc.json`:
```js
{
	"extends": "mudamuda"
}
```
Extended rule sets are also available to JavaScript frameworks & environments:
```js
{
	"extends": "mudamuda/node"
}
```
